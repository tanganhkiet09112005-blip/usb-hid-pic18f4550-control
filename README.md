# 🔌 USB HID Communication Lab (PIC18F4550 & C#)

Research and implementation project of the USB HID protocol to control and monitor peripheral devices from a computer.

## 🛠️ Technical Components
* **Microcontroller:** PIC18F4550 (Supports USB 2.0 full-speed).
* **Firmware Language:** C (MikroC PRO for PIC).
* **Software Language:** C# (.NET WinForms).
* **Simulation Tool:** Proteus 8.

## 🚀 Main Features
* **Plug-and-Play:** The computer automatically recognizes the device according to the HID standard (no driver needed).
* **Real-time Monitoring:** Updates the status of buttons and counters from hardware to software via interrupt mechanism.
* **Remote Control:** Control LED/Relay states directly from the computer interface.
* **USB Buffering:** Use a 64-byte buffer for accurate and stable data transmission.
